Architecture

Browser → CloudFront CDN → S3 Bucket

This page is being served from an Amazon S3 bucket through Amazon CloudFront, a global content delivery network.

CloudFront caches the content at edge locations worldwide, reducing latency for users regardless of their geographic location.

The S3 bucket has all public access blocked and is only accessible via Origin Access Control (OAC), ensuring secure content delivery.
